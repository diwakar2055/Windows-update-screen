🎯 For a True Prank: Disable System Keys

To prevent exiting the simulation with keys like Esc, Alt+Tab, or the Windows Key, you need to temporarily disable them at the operating system level. This website cannot do this for security reasons.

We recommend SharpKeys, a free and safe tool for Windows that remaps keys by changing a simple registry setting.

📋 How to Use SharpKeys:

  1. Download and install SharpKeys from GitHub.
  2. Open SharpKeys and click "Add".
  3. In the left column ("Map this key"), select the key you want to disable (e.g., "Special: Escape").
  4. In the right column ("To this key"), select the very first option: "Turn Key Off".
  5. Click "OK" and repeat for other keys like Alt, Tab, and the Windows Key.
  6. Once you've added all the keys, click "Write to Registry" and reboot your computer.

🖱️ Disable Touchpad Gestures (Important!)

Don't forget to disable touchpad gestures that could exit fullscreen:

  1. Go to Windows Settings → Devices → Touchpad
  2. Scroll down to "Three-finger gestures" and "Four-finger gestures"
  3. Set all gesture actions to "Nothing" or disable them completely
  4. This prevents accidental exits via touchpad swipes

🔄 To re-enable your keys, just open SharpKeys again, delete the mappings you created, and click "Write to Registry".
